Quickly create multiple posts, pages, or other custom post types from a single interface. Helpful for WordPress developers.

== Description ==

As a WordPress developer you may want to quickly create multiple posts en masse. Use the bulk content creator to create multiple posts, pages, or other custom post types from a single interface. You may also assign featured images to a post type.

= Issues =

See an problem? Want a feature? Log it on [github.com](https://github.com/luk3thomas/wpBulkContentCreator/issues).

== Installation ==

1. Verify that you have PHP5, which is required for this plugin.
1. Upload the whole `bulk-content-creator` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In WordPress admin go to 'Tools' -> 'Bulk Content Creator'
